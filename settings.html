Why GitHub sync?
Your workout data is stored in your GitHub repository, ensuring it's never lost even if iOS clears the app cache. Your data stays safe and you stay in control.

Setup Instructions

  1. Go to GitHub Settings → Personal Access Tokens
  2. Click "Generate new token" (fine-grained)
  3. Set repository: AbracadabraApp/workout-tracker
  4. Grant permissions: Contents → Read & Write
  5. Copy the token and paste it above
⚠️ Security Note:
Your token is stored locally in this app only. Never share your token with anyone. You can revoke it anytime in your GitHub settings.
